Image Tagging Instructions
   

You will be shown an image.
The task is to select all the tags that describe the contents/objects in the image.
If there are multiple tags in a row, select the most specific one.

Multiple tags in a row: Rows with multiple tags present the hierarchical relation of words. The rightmost word is the most general term of words to its left. (For example, rocking chair is a chair.) Select the most specific one in each row.

None of the above: Select "none of the above" when none of the tags describes the objects in the image.
